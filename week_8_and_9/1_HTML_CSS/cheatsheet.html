Ruby Hashes, Arrays, Strings Integers

Hashes

A collection of key-value pairs. Can be created two ways:
hash1 = {"key1"=>value, "key2"=>value}
hash2 = Hash.new
hash2["key1"]=value

Arrays

Similar to hashes, arrays are also ordered collections that are indexed by integer. However, in the case of an array, there is only one object associated with each element (as opposed to a key and value).
Arrays can be created in two ways: array=Array.new
array=[1,2,3] There are many methods available in Ruby that allow us to access and manipulate data within arrays.

Strings and Integers

Ruby strings hold text. For example, "Hello world." is a string. You can use the following syntax to insert code into a string. "This string now contains the number four #{2+2}" Strings can be accessed by index. They can also be concatenated using the "+" operator.

Integers

Numbers in ruby are categorized as either integers or floats. Floats represent inexact numbers, while integers represent whole numbers.
